# Research

**Status:** Non-normative workspace
**Authority:** Evidence and investigation only
**Owner:** Founding Team
**Governing reference:** [Architecture authority map](../architecture/README.md)

This directory contains investigations, experiments, threat studies, provider
comparisons, and source material.

Research is evidence, not a decision. A conclusion changes architecture only
through an accepted ADR and incorporation into the Architecture Freeze.

Each research note SHOULD record:

- question and scope;
- date and author;
- sources and methodology;
- observations separated from interpretation;
- limitations and unresolved questions;
- proposed follow-up or ADR, if any.
